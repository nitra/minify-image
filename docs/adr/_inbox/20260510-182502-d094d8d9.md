**Рішення/Процедура/Факт:**

- v4 змінює дефолти `preset-default`: `removeViewBox` та `removeTitle` тепер **вимкнені** (раніше ввімкнені). Проєкт не перевизначав ці плагіни — перейшов на безпечніший дефолт (зберігає `viewBox` і `<title>`).

## ADR Захисний дефолт: src-tauri/icons виключається з обробки

**Контекст:** Tauri-проєкти зберігають PNG-іконки у `src-tauri/icons/` — канонічний шлях, генерований `tauri icon` CLI. Квантизація RGBA-PNG у palette (яку sharp застосовує по-замовчуванню) ламає `tauri::generate_context!` (panic «icon … is not RGBA»). Користувачі не повинні знати про це і явно передавати `--ignore=src-tauri/icons/**`.

**Рішення/Процедура/Факт:**
- `**/src-tauri/icons/**` додано до always-on exclude в `globOptions.ignore` (`npm/src/index.js:78-91`).
- `src-tauri/icons` додано у regex `AVIF_IGNORE_PATH_RE` (`npm/src/index.js:376-388`) — друга лінія оборони (AVIF-генерація там теж безглузда).
- Оновлено HELP_TEXT для `--ignore` (`npm/src/index.js:39-41`).
- Нові regression-тести в `demo/test/tauri-icons-default-ignore.test.js`: RGBA-PNG під `src-tauri/icons/` лишається байт-у-байт; `--avif` не пише `.avif` поряд.
- README (npm): нова секція «Tauri / icon assets» з поясненням причини.
- CHANGELOG: `[3.6.0] - 2026-05-10`, версія `3.5.0 → 3.6.0`.
- `.cspell.json`: додано `tauri`.

**Обґрунтування:** Tauri фіксує цей шлях як канонічну локацію іконок — шанс false-positive мінімальний, а benefit — нульова регресія для Tauri-проєктів без жодних додаткових опцій. Scope навмисно обмежений тільки `src-tauri/icons/**` (не `app-icons/`, `electron/build/icons/` тощо).

**Розглянуті альтернативи:**
1. `--preserve-color-type=<glob>` CLI-прапорець + per-package config `{ "@nitra/minify-image": { "preserve-color-type": [...] } }` — обговорювались у spec, але відкинуті: складніший API без доведеної потреби, Tauri покривається дефолтом.
2. Розширити на інші electron/app-icons layouts — відкладено як окремий issue.

**Зачіпає:** `npm/src/index.js` (globOptions.ignore, AVIF_IGNORE_PATH_RE, HELP_TEXT), `npm/README.md`, `npm/CHANGELOG.md`, `npm/package.json`, `demo/test/tauri-icons-default-ignore.test.js`, `.cspell.json`.
